## Electron + Vite + TypeScript + Tailwind CSS Starter

This is a professional, high-performance **Electron** boilerplate designed for modern desktop application development. It comes pre-configured with **Vite**, **TypeScript**, and **Tailwind CSS**.

## Key Features

- **Vite:** Lightning-fast HMR (Hot Module Replacement).
- **TypeScript:** Type safety for both Main and Renderer processes.
- **Tailwind CSS:** Utility-first CSS framework for rapid UI development.
- **Electron Builder:** Pre-configured for packaging Windows, Mac, and Linux apps.
- **IPC Communication:** Example of secure bridge between Main and Renderer processes.

## Tech Stack

- **Framework:** Electron
- **Bundler:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Getting Started

Follow these steps to get the project running on your local machine:

### 1. Clone the repository

```bash
git clone https://github.com/adnanhjoy/electron-vite-ts-boilerplate.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

### Building for Production

To package the application for different operating systems, use the following commands:

### 4. For windows

```bash
npm run build:win
```

### 5. For mac

```bash
npm run build:mac
```

### 6. For linux

```bash
npm run build:linux
```
